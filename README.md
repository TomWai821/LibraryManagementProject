This e-Libaray managment project made with html+css+boostrap(frontend), ASP.NET(backend) and MSSQL(database)

And there are 3 pages for user(login page, page for view books and page for check user profile), 5 pages for admin(login page, author management page, user management page, book management page and book issuing page)

Using ASP.NET sql api communicate with MSSQL and complete all the CRUD operations in all management pages

Using html,css and bootstrap to setup UI(Card view, navbar, button)
